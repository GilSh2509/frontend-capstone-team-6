# frontend-capstone-team-6 

## תיאור המערכת
מערכת זו פותחה עבור חברה בתחום שירות הלקוחות והמוצרים, במטרה להוות פלטפורמה אינטראקטיבית לניהול ושיתוף חוות דעת. המערכת מאפשרת למשתמשים להביע את דעתם על מגוון פריטים, שירותים ומוצרים בצורה ידידותית ונוחה.

## טכנולוגיות
בפרויקט זה השתמשנו בטכנולוגיות הבאות:
* **HTML5** & **CSS3** ליצירת המבנה והעיצוב.
* **JavaScript** ללוגיקה ודינמיות של האתר.
* **GitHub** לניהול גרסאות ועבודה צוותית.

## סיכום פיצ'רים
* **דף בית אינטראקטיבי:** הצגת מידע כללי ותמונות, הצגת הפריטים הכי מומלצים, אפשרויות סינון וחיפוש.
* **טופס ביקורת:** אפשרות למשתמשים להשאיר חוות דעת עם כוכבי דירוג וטקסט חופשי.
* **רשימת פריטים:** תצוגה מסודרת של מוצרים או שירותים לאחר סינון שנבחר בדף הבית.
* **פרטי פריט:** תצוגה של מוצר ספציפי שנבחר מהרשימה, כולל הביקורות עליו, תמונות, אפשרות לדירוג והשארת ביקורת. 
* **עיצוב רספונסיבי:** התאמה מלאה למכשירים ניידים ומחשבים.

   הריצו את הפקודה הבאה בטרמינל:
